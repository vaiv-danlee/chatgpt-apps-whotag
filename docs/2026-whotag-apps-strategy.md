## 1. 문제 인식

### AI 시대의 정보 과부하

> *"빠르게 원하는 인플루언서를 찾고 춤춰라!"*
> 과연... 춤을 출 수 있을까?

현재 사용자들이 겪는 문제:

| 문제 | 사용자 반응 |
|------|------------|
| 정보 과다 | "100명, 2000명 많이 보여주면 뭐해. 언제 다 읽고 걸러내라는거야" |
| 비용 부담 | "부스터가 24시간에 200달러? 하루종일 글만 읽다 끝나겠네" |
| 복잡한 UI | AI 시대, 숏폼 시대에 화면 가득한 텍스트는 숨이 턱- 막힘 |

**핵심 인사이트**: 좋은 데이터를 제공하는 것만으로는 부족하다. 사용자가 **소화할 수 있는 형태**로 전달해야 한다.

## 2. 사용자 시나리오

### 시나리오 1: whotag.ai & ChatGPT 하이브리드

```
┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
│   whotag.ai     │ ───▶ │   ChatGPT +     │ ───▶ │   최종 결과물    │
│   검색 & 탐색    │      │   WHOTAG Apps   │      │   버킷 B        │
└─────────────────┘      └─────────────────┘      └─────────────────┘
```

1. **whotag.ai에서 자연어 검색** → 인플루언서 탐색
2. **프로필/피드 훑어보며** → 버킷 A에 담기
3. **ChatGPT + WHOTAG Apps 전환** → 버킷 A 불러오기
4. **세부 조건으로 필터링 요청**
   - *"Fwee 브랜드 특성 고려해서 가장 적합한 20명 선별해줘"*
5. **GPT가 정보를 요리해서 제공**
6. **후속 작업 요청**
   - 브랜드 메시지 시안 작성 (현지어 + 영어)
   - 컨택 메일 초안 작성
7. **최종 선별 → 버킷 B 저장**

### 시나리오 2: ChatGPT 솔로 플레이

```
┌─────────────────────────────────────────────────────────────────┐
│                    ChatGPT + WHOTAG Apps                        │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐     │
│  │  검색    │ → │  탐색    │ → │  선별    │ → │ 후속작업  │     │
│  └──────────┘   └──────────┘   └──────────┘   └──────────┘     │
└─────────────────────────────────────────────────────────────────┘
```

1. **ChatGPT에서 WHOTAG Apps 연동** → 자연어 검색
2. **Apps UI로 프로필/피드 탐색** → 버킷에 담기 (Optional)
   - 이미지 전달 가능 시, 피드 느낌 기반 GPT 선별도 가능 (실험 필요)
3. **버킷 인플루언서 불러오기** (Optional)
4. **시나리오 1의 4-7 과정과 동일**

## 3. 역할 정의

| 구성 요소 | 역할 | 특징 |
|-----------|------|------|
| **WHOTAG** | 고품질 글로벌 뷰티 인플루언서 데이터베이스 | 핵심 자산 |
| **ChatGPT** | 분석왕 똑똑이 | 정보 가공 & 인사이트 도출 |
| **Apps** | WHOTAG ↔ ChatGPT 브릿지 | 서비스 확장 도구 |

### Apps의 본질

> Apps가 별거냐? **별거 아님**.
> MCP + UI. MCP보다 좀 더 fancy한 느낌. 인터랙티브하게 보여주니까.

**확장 가능성**: MCP 기반으로 다른 호스트(Claude, Gemini 등)에도 **확장팩처럼** 연결 가능

## 4. 개발 범위

### 현재 범위: 인플루언서 검색 도구

- 자연어 기반 인플루언서 검색
- 프로필 및 피드 탐색
- 버킷 관리

### 확장 가능 범위: 글로벌 뷰티 시장 분석

WHOTAG의 빅쿼리에는 **수억 건**의 데이터가 축적되어 있음:
- Profile 데이터
- Media 데이터
- Inferred 데이터

**이를 활용한 파생 도구 개발이 충분히 가능**

## 5. 파생 도구 아이디어

### 5.1 트렌드 분석 도구

| 도구명 | 설명 | 예시 쿼리 |
|--------|------|----------|
| `analyze_beauty_trends` | 특정 기간/지역의 뷰티 트렌드 분석 | "2024년 일본에서 가장 많이 언급된 스킨케어 성분은?" |
| `track_hashtag_growth` | 해시태그 성장률 추적 | "#glassskin 해시태그의 6개월 성장 추이" |
| `compare_regional_trends` | 지역별 트렌드 비교 | "한국 vs 미국의 립 메이크업 트렌드 차이" |

### 5.2 경쟁사/브랜드 분석 도구

| 도구명 | 설명 | 예시 쿼리 |
|--------|------|----------|
| `analyze_brand_mentions` | 브랜드 언급량 및 감성 분석 | "에스티로더 vs 랑콤의 인플루언서 언급 비교" |
| `find_brand_ambassadors` | 브랜드 협업 인플루언서 탐지 | "샤넬 뷰티와 협업한 인플루언서 목록" |
| `track_product_launches` | 신제품 출시 반응 추적 | "디올 립글로우 신제품 인플루언서 반응" |

### 5.3 시장 인사이트 도구

| 도구명 | 설명 | 예시 쿼리 |
|--------|------|----------|
| `market_size_estimate` | 카테고리별 시장 규모 추정 | "동남아 클린뷰티 시장의 인플루언서 활동량" |
| `emerging_markets` | 성장하는 시장/카테고리 탐지 | "최근 급성장 중인 뷰티 서브카테고리" |
| `audience_demographics` | 타겟 오디언스 분석 | "비건 뷰티 콘텐츠 소비자 연령대 분포" |

### 5.4 콘텐츠 분석 도구

| 도구명 | 설명 | 예시 쿼리 |
|--------|------|----------|
| `top_performing_content` | 고성과 콘텐츠 패턴 분석 | "뷰티 튜토리얼 중 조회수 높은 포맷은?" |
| `content_calendar_insights` | 최적 포스팅 시점 분석 | "뷰티 콘텐츠 최적 업로드 요일/시간" |
| `viral_content_patterns` | 바이럴 콘텐츠 특성 분석 | "최근 바이럴된 뷰티 콘텐츠 공통점" |

## 7. 결론: 확장 vs 브랜딩

### 답: **확장 >>> 브랜딩**

```
                    ┌─────────────────────────────────┐
                    │     사용자 접점 레이어          │
                    │  (ChatGPT Apps, 웹서비스, 앱)   │
                    └─────────────────────────────────┘
                                    │
                                    ▼
                    ┌─────────────────────────────────┐
                    │         WHOTAG 레이어           │
                    │   (고품질 데이터베이스 + API)    │
                    └─────────────────────────────────┘
```

### 핵심 전략

| 항목 | 내용 |
|------|------|
| **사용자 인지** | WHOTAG이 뭔지 몰라도 됨 |
| **우리의 역할** | 고품질 데이터만 잘 제공 |
| **포지셔닝** | WHOTAG은 하위 레이어에 존재 |
| **확장 방향** | WHOTAG 위에 다양한 서비스가 올라감 |

### 기대 효과

```
확장 → 인지도 상승 → 자연스러운 브랜딩
```

**궁극적 목표**: 타 서비스들이 *"WHOTAG의 데이터를 소싱해 온다"*고 광고하는 것
