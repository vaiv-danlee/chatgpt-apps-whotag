## 보고서 유형별 구조

### 1. 시장 트렌드 리포트 (8-10장)

```
1. 표지
2. Executive Summary
3. 시장 개요 (국가/지역 컨텍스트)
4. 해시태그 트렌드 TOP 10
5. 카테고리별 트렌드 분석
6. 성장 트렌드 (시계열)
7. 주요 발견사항
8. 권장사항
9. 부록 (선택)
```

### 2. 인플루언서 캐스팅 리포트 (8-12장)

```
1. 표지
2. Executive Summary
3. 검색 기준 및 방법론
4. 추천 인플루언서 TOP 5 (각 1장 또는 비교표)
5. 상세 프로필 분석 (2-4장)
6. 비교 분석 테이블
7. 캐스팅 추천 및 우선순위
8. 예상 효과 및 KPI
9. 부록: 전체 후보 리스트
```

### 3. 시장조사 리포트 (10-15장)

```
1. 표지
2. Executive Summary
3. 목차
4. 조사 배경 및 목적
5. 시장 개요
6. 주요 트렌드 분석 (2-3장)
7. 경쟁 환경 분석
8. 인플루언서 생태계
9. 기회 영역
10. 권장 전략
11. 다음 단계
12. 부록
```

## 슬라이드 유형별 템플릿

### 표지 (Cover Slide)

```
┌─────────────────────────────────────┐
│                                     │
│         [로고 또는 아이콘]           │
│                                     │
│     보고서 제목 (28-36pt Bold)       │
│     부제목 (18-24pt)                 │
│                                     │
│     날짜 | 작성자/팀                 │
│                                     │
└─────────────────────────────────────┘
```

**색상**: Navy 배경 + White 텍스트 또는 White 배경 + Navy 텍스트

### Executive Summary

```
┌─────────────────────────────────────┐
│ Executive Summary                   │
├─────────────────────────────────────┤
│                                     │
│  핵심 발견 1                         │
│  • 상세 설명 (1-2줄)                 │
│                                     │
│  핵심 발견 2                         │
│  • 상세 설명 (1-2줄)                 │
│                                     │
│  핵심 발견 3                         │
│  • 상세 설명 (1-2줄)                 │
│                                     │
└─────────────────────────────────────┘
```

**규칙**:
- 핵심 발견 3개 이내
- 각 발견은 숫자 또는 아이콘으로 구분
- 정량적 데이터 포함 권장 (예: "팔로워 100만 이상 인플루언서 15명 발굴")

### 데이터 차트 슬라이드

**Two-Column Layout (권장)**:
```
┌─────────────────────────────────────┐
│ 슬라이드 제목                        │
├──────────────────┬──────────────────┤
│                  │                  │
│  핵심 인사이트    │    [차트]        │
│  • 포인트 1      │                  │
│  • 포인트 2      │                  │
│  • 포인트 3      │                  │
│                  │                  │
│                  │   출처: whotag   │
└──────────────────┴──────────────────┘
```

**Full-Width Chart Layout**:
```
┌─────────────────────────────────────┐
│ 슬라이드 제목                        │
├─────────────────────────────────────┤
│                                     │
│              [차트]                  │
│                                     │
│                                     │
│                                     │
│                       출처: whotag   │
└─────────────────────────────────────┘
```

### 인플루언서 프로필 카드

**단일 프로필 (이미지 포함)**:
```
┌─────────────────────────────────────┐
│ 추천 인플루언서 #1                   │
├─────────────────────────────────────┤
│  ┌────────┐                         │
│  │ 프로필 │  @username              │
│  │  사진  │  팔로워: 1.2M           │
│  └────────┘  참여율: 4.5%           │
│                                     │
│  주요 카테고리: 뷰티, 라이프스타일   │
│  대표 콘텐츠: ...                    │
│  추천 이유: ...                      │
│                                     │
└─────────────────────────────────────┘
```

**프로필 이미지 처리 워크플로우**:
1. `get_profile_image` MCP 툴 호출 (userId 전달)
2. 반환된 이미지 데이터를 로컬 파일로 저장 → `./temp/profile_{userId}.jpg`
3. HTML에서 로컬 경로 참조: `<img src="./temp/profile_{userId}.jpg">`
4. html2pptx 변환 완료 후 `./temp/` 폴더 정리

**비교 테이블**:
```
┌─────────────────────────────────────┐
│ 인플루언서 비교 분석                 │
├─────────────────────────────────────┤
│ ┌──────┬──────┬──────┬──────┬─────┐ │
│ │      │ 팔로워│ ER  │카테고리│점수│ │
│ ├──────┼──────┼──────┼──────┼─────┤ │
│ │@user1│ 1.2M │ 4.5%│ 뷰티  │ 95 │ │
│ │@user2│ 800K │ 5.2%│ 패션  │ 92 │ │
│ │@user3│ 2.1M │ 3.1%│ 음식  │ 88 │ │
│ └──────┴──────┴──────┴──────┴─────┘ │
└─────────────────────────────────────┘
```

## 레이아웃 패턴

### 비율 가이드

| 요소 | 권장 비율 |
|------|-----------|
| 제목 영역 | 10-15% |
| 본문 영역 | 75-85% |
| 푸터/출처 | 5-10% |

### 여백 규칙

- **외부 여백**: 0.5인치 (36pt)
- **요소 간 간격**: 12-24pt
- **텍스트 줄 간격**: 1.2-1.5배

### 정보 밀도

| 슬라이드 유형 | 최대 요소 수 |
|---------------|--------------|
| 표지 | 3-4개 |
| 텍스트 중심 | 불릿 5개 이내 |
| 차트 중심 | 차트 1개 + 불릿 3개 |
| 비교 테이블 | 5-7행 x 4-6열 |
| 프로필 카드 | 1-3개 |

### 프로필 이미지 스타일

```css
.profile-image {
  width: 80px;
  height: 80px;
  border-radius: 50%;
  object-fit: cover;
  border: 2px solid #1E3A5F;
}
```
